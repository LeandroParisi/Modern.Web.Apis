Ambient Light Events

The AmbientLightSensor interface returns the current light level or illuminance of the ambient light around the hosting device..
It allows them to react to such a change, for example by changing the color contrast of the User Interface (UI) or by changing the exposure necessary to take a picture.

warning Still experimental Enable the generic sensor extra classes flag in Chrome (chrome://flags/#enable-generic-sensor-extra-classes) to test it.
