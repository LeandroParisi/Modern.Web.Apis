Native File System API

The Native File System API provides new features to manipulate local files, providing an improved experience to our users similar to a native app. FOr example we do not have to download again the file each time we want to edit it, but we can save it directly or even implement auto save funtionalitiy.

lock Feature available only in secure contexts (HTTPS).

warning Still experimental Enable the Native File System flag in Chrome (chrome://flags/#native-file-system-api) to test it. Active by default from Chrome 86.
