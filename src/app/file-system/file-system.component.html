Native File System API

The Native File System API offers new features to manipulate local files providing an improved experience to our users similar to a native app. We don't have to download the file each time we want to edit it, but we can save it directly. We can even implement an auto-save functionality.

warning Still experimental Enable the Native File System flag in Chrome (chrome://flags/#native-file-system-api) to test it. Active by default from Chrome 86.
